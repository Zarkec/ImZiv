# 图片工作流功能说明

图片工作流用于把图片处理步骤组织成节点图。节点之间通过输入、输出连接传递图片集、裁切区域等数据，最后使用预览或保存节点作为终端节点执行。

## 基本操作

- 打开图片工作流视图后，在节点编辑区右键添加节点。
- 从输出端拖到输入端建立连接。
- 选中节点后，在节点内部或右侧属性区调整参数。
- 重复添加同类节点时，节点标题会自动编号，方便区分多个分支。
- 使用工具栏的新建、打开、保存、另存为管理工作流文件。
- 使用运行按钮执行当前工作流，运行期间可以停止。
- 批量处理或耗时任务在后台执行，界面会显示进度。

## 数据约定

- 图片数据使用 OpenCV 的 `cv::Mat` 保存。
- 文件解码后的 3 通道图片默认按 OpenCV 习惯解释为 BGR。
- 色彩空间转换节点会默认把 3 通道输入视为 BGR，再转换到目标空间。
- 通道分离节点不会判断色彩空间，只按当前 `cv::Mat` 的通道顺序拆分。

## 输入节点

### 当前图片

读取主图片查看器当前打开的图片，并输出为图片集。

适合处理单张当前图片。

### 图片文件

从指定图片路径读取单张图片，并输出为图片集。

参数：

- 路径：图片文件路径，可以手动输入，也可以点击按钮选择图片。

适合在不切换主图片查看器当前图片的情况下，固定读取某一张图片。

### 图片目录

从指定目录读取图片，并输出为图片集。

参数：

- 目录：图片所在目录。
- 递归：是否递归读取子目录。

支持格式：

- `png`
- `jpg` / `jpeg`
- `bmp`
- `tiff` / `tif`
- `webp`
- `gif`

## 几何变换节点

### 裁切

按矩形区域裁出图片。

输入：

- 图片

参数：

- `X`：左上角 X 坐标，最小值 `0`。
- `Y`：左上角 Y 坐标，最小值 `0`。
- `W`：宽度，最小值 `1`。
- `H`：高度，最小值 `1`。

辅助操作：

- 使用当前图片尺寸：把区域设为整张当前图片。
- 自由框选：在主图片查看器中框选裁切区域。

输出：

- 裁切后的图片

如果裁切区域超出图片范围，执行会失败并提示错误。

### 缩放

调整图片尺寸。

参数：

- 宽度：目标宽度。填 `0` 表示按高度或比例自动计算。
- 高度：目标高度。填 `0` 表示按宽度或比例自动计算。
- 比例：当宽度和高度都为 `0` 时使用，必须大于 `0`。
- 插值：缩放采样方式，可选 `线性`、`最近邻`、`面积`、`三次`。

规则：

- 宽度和高度都大于 `0`：直接缩放到指定尺寸。
- 只设置宽度：按原图比例自动计算高度。
- 只设置高度：按原图比例自动计算宽度。
- 宽度和高度都为 `0`：按比例缩放。
- 通常缩小时用 `面积`，放大时用 `线性` 或 `三次`。

### 旋转

按角度旋转图片。

参数：

- 角度：旋转角度，单位为度。
- 扩展画布：开启后会扩大输出尺寸，尽量保留旋转后的完整图片；关闭后保持原尺寸，边缘可能被裁掉。
- 插值：旋转采样方式，可选 `线性`、`最近邻`、`面积`、`三次`。

说明：

- 空白区域使用黑色填充。
- 正角度按 OpenCV 的坐标系旋转。

### 翻转

按方向翻转图片。

参数：

- 方向：可选 `水平`、`垂直`、`水平+垂直`。

### 平移

按像素移动图片内容。

参数：

- `X`：水平移动像素。正数向右，负数向左。
- `Y`：垂直移动像素。正数向下，负数向上。

说明：

- 输出尺寸保持不变。
- 移出画布的内容会被裁掉，空白区域使用黑色填充。

## 颜色处理节点

### 灰度

把图片转换为灰度图。

规则：

- 1 通道输入：复制原图。
- 3 通道输入：按 BGR 转灰度。
- 4 通道输入：按 BGRA 转灰度。

说明：

- 这个节点等价于“色彩空间转换”里选择“灰度”，只是作为常用操作单独保留。

### 色彩空间转换

把图片转换到指定色彩空间。

目标：

- `BGR`
- `RGB`
- `灰度`
- `HSV`
- `HLS`
- `Lab`
- `YCrCb`

说明：

- 3 通道输入默认按 BGR 解释。
- 4 通道输入会先按 BGRA 转为 BGR。
- 1 通道输入在转换到彩色空间时会先转为 BGR。

### 通道分离

把输入图片拆成三个单通道输出。

输出：

- 通道 1
- 通道 2
- 通道 3

通道含义取决于输入图片当前的色彩空间：

- BGR 输入：通道 1/2/3 分别是 B/G/R。
- RGB 输入：通道 1/2/3 分别是 R/G/B。
- HSV 输入：通道 1/2/3 分别是 H/S/V。
- Lab 输入：通道 1/2/3 分别是 L/a/b。
- YCrCb 输入：通道 1/2/3 分别是 Y/Cr/Cb。

如果输入是灰度图，三个输出都会使用同一个灰度通道，便于保持固定输出端数量。

### 通道合并

把三个单通道输入合并为一张图片。通道分离的逆操作。

输入：

- 通道 1
- 通道 2
- 通道 3

输出：

- 图片

规则：

- 三个输入的图片数量必须一致。
- 三个通道的尺寸必须一致。

### 掩膜应用

把掩膜应用到图片上，掩膜为 `0` 的像素会被置为黑色（或透明）。

输入：

- 图片
- 掩膜

说明：

- 图片和掩膜的数量、尺寸必须一致。
- 掩膜会自动转为单通道。如果掩膜是多通道，会先转灰度。
- 对于 BGR 输入，掩膜为 `0` 的区域变为黑色。
- 对于 BGRA 输入，掩膜会作为 Alpha 通道写入，实现透明效果。

使用场景：

- HSV 范围阈值 → 掩膜应用：按颜色提取目标区域。
- Canny 边缘检测 → 掩膜应用：只保留边缘位置的原图像素。

### 图片混合

把两张图片按混合模式叠加。

输入：

- 底层图片
- 上层图片

参数：

- 底层不透明度：底层图片的可见程度，范围 `0-1`，默认 `0.5`。
- 上层不透明度：混合效果的可见程度，范围 `0-1`，默认 `0.5`。
- 混合模式：可选 `正常`、`正片叠底`、`滤色`、`叠加`、`柔光`、`差值`。

混合模式说明：

- 正常：线性 alpha 混合。
- 正片叠底：`底层 × 上层`，结果偏暗。
- 滤色：`1 - (1-底层) × (1-上层)`，结果偏亮。
- 叠加：暗区正片叠底、亮区滤色，增强对比。
- 柔光：类似叠加但效果更温和。
- 差值：`|底层 - 上层|`，用于差异检测。

规则：

- 尺寸不一致时，上层会自动缩放到底层尺寸。
- 灰度和 4 通道输入会统一转为 3 通道处理。
- 底层图片的 Alpha 通道会被保留。

## 色彩调节节点

### 亮度/对比度

调整图片的亮度和对比度。

参数：

- 亮度：范围 `-100` 到 `100`。
- 对比度：范围 `-100` 到 `100`。

说明：

- 亮度直接加减像素值。
- 对比度正值按 1 + contrast/50 放大与 128 的偏差，负值按 1 + contrast/100 缩小偏差。
- 支持 1/3/4 通道。

### 色相/饱和度/明度

在 HSV 空间调整色相、饱和度和明度。

参数：

- 色相偏移：范围 `-180°` 到 `180°`。
- 饱和度：乘数，范围 `0` 到 `3`，`1` 为不变。
- 明度：乘数，范围 `0` 到 `3`，`1` 为不变。

说明：

- 灰度图只受明度影响。
- 色相偏移会自动取模，不会超出范围。
- 4 通道输入的 Alpha 通道不受影响。

### 反相

反转图片的每个像素值（255 - 原值）。

说明：

- 等价于 Photoshop 的"反相"。
- 支持 1/3/4 通道。

### 直方图均衡化

增强图片对比度。

说明：

- 灰度图直接均衡化。
- 彩色图转 YCrCb 空间，仅均衡化 Y（亮度）通道，再转回 BGR，避免色彩偏移。
- 4 通道输入保留 Alpha 通道。

### Gamma 校正

通过 Gamma 曲线调整图片明暗。

参数：

- Gamma：范围 `0.1` 到 `10`，`1` 为不变。
  - `> 1`：图片变亮（提亮暗部）。
  - `< 1`：图片变暗（压暗亮部）。

说明：

- 使用 LUT 查找表：`output = 255 × (input/255)^(1/gamma)`。
- 支持 1/3/4 通道，Alpha 通道不受影响。

### 色阶

通过黑场、白场和中间调映射图片色调范围。

参数：

- 黑场：输入黑色点，范围 `0-254`，默认 `0`。
- 白场：输入白色点，范围 `黑场+1` 到 `255`，默认 `255`。
- 中间调：中间调 Gamma，范围 `0.1-10`，默认 `1`。
  - `< 1`：中间调偏亮。
  - `> 1`：中间调偏暗。

说明：

- 先将像素从 `[黑场, 白场]` 线性映射到 `[0, 255]`，再应用 Gamma 中间调校正。
- 支持 1/3/4 通道，Alpha 通道不受影响。

## 阈值分割节点

### 阈值二值化

把图片转换成二值图。

参数：

- 阈值：范围 `0-255`。

规则：

- 彩色图会先转灰度。
- 大于阈值的像素变为 `255`，其他变为 `0`。

### Otsu 二值化

使用 Otsu 方法自动计算阈值并生成二值图。

规则：

- 彩色图会先转灰度。
- 不需要手动设置阈值。

### 自适应二值化

按局部区域动态计算阈值并生成二值图。

参数：

- 块大小：最小值 `3`，自动修正为奇数。
- `C`：范围 `-255` 到 `255`。

规则：

- 彩色图会先转灰度。
- 使用 OpenCV 的 `ADAPTIVE_THRESH_GAUSSIAN_C`。

### HSV 范围阈值

根据 HSV 范围生成二值掩膜。

参数：

- 输入已是 HSV：如果前面已经接了色彩空间转换到 HSV，则勾选此项。
- HSV 容差：从采样点生成范围时，在 H/S/V 三个方向额外扩张的范围。
- HSV 下限：三个值依次为 H/S/V 的下限。
- HSV 上限：三个值依次为 H/S/V 的上限。
- H 范围：`0-179`。
- S/V 范围：`0-255`。
- 开始选点：进入图片取样模式，在查看器里左键点击多个目标颜色点。
- 重新选点：选点过程中再次点击按钮，会清空当前采样点并重新开始。
- 右键结束：根据所有采样点的最小/最大 HSV 值，再叠加 HSV 容差，自动写入 HSV 最小/最大。
- Ctrl+Z：选点过程中撤销最后一个采样点。

输出：

- 掩膜：范围内像素为 `255`，范围外像素为 `0`。

使用建议：

- 如果输入是原图，可以直接接 HSV 范围阈值节点，不需要先接色彩空间转换。
- 如果前面已经显式转换为 HSV，应勾选“输入已是 HSV”，避免重复转换。
- OpenCV 中 H 的范围是 `0-179`，不是常见的 `0-360`。
- 采样点越覆盖目标颜色的亮暗变化，生成的范围越稳；背景颜色不要点进去，否则阈值会被放宽。
- H 当前按普通最小/最大值生成范围，不做跨 `0/179` 的环形拆分。提取红色这类可能跨边界的颜色时，必要时需要拆成两个 HSV 范围分别处理。

## 形态学操作节点

### 形态学操作

对图片执行形态学运算。

参数：

- 操作：可选 `腐蚀`、`膨胀`、`开运算`、`闭运算`、`形态学梯度`、`顶帽`、`黑帽`。
- 核形状：可选 `矩形`、`椭圆`、`十字`。
- 核宽度：结构元素宽度，最小值 `1`，自动修正为奇数。
- 核高度：结构元素高度，最小值 `1`，自动修正为奇数。
- 迭代次数：最小值 `1`。

说明：

- 支持任意通道数的输入。
- `开运算` = 先腐蚀再膨胀，用于去除小噪点。
- `闭运算` = 先膨胀再腐蚀，用于填补小孔洞。
- `形态学梯度` = 膨胀 − 腐蚀，用于提取边缘。
- `顶帽` = 原图 − 开运算，用于提取亮细节。
- `黑帽` = 闭运算 − 原图，用于提取暗细节。

## 平滑处理节点

### 均值滤波

使用归一化盒滤波器对图片做平滑。

参数：

- 核宽度：自动修正为奇数，最小值 `1`。
- 核高度：自动修正为奇数，最小值 `1`。

### 高斯滤波

使用高斯核加权平均对图片做平滑。

参数：

- 核宽度：自动修正为奇数，最小值 `1`。
- 核高度：自动修正为奇数，最小值 `1`。
- Sigma X：水平方向标准差，`0` 时根据核宽度自动计算。
- Sigma Y：垂直方向标准差，`0` 时根据核高度自动计算。

### 中值滤波

使用邻域中值替代中心像素，对椒盐噪声效果好。

参数：

- 核大小：单个奇数值，自动修正，最小值 `1`。

### 双边滤波

在平滑的同时保留边缘。

参数：

- 邻域直径：滤波时考虑的邻域大小。
- Sigma 颜色：颜色空间的标准差，值越大颜色差异的影响范围越宽。
- Sigma 空间：坐标空间的标准差，值越大越远的像素也会互相影响。

## 边缘检测节点

### Canny 边缘检测

使用 Canny 算法检测图片边缘。

参数：

- 低阈值：低于此值的梯度被抑制。
- 高阈值：高于此值的梯度被确认为边缘。高低阈值之间的梯度仅在连接到高阈值边缘时保留。
- 孔径大小：Sobel 算子的核大小，步长为 `2`，自动修正为奇数。

说明：

- 彩色图会先转灰度。
- 输出为单通道二值边缘图。

### Sobel 算子

使用 Sobel 算子计算图片梯度。

参数：

- 方向：可选 `X 方向`、`Y 方向`、`XY 方向`。XY 方向会同时计算两个方向的梯度。
- 核大小：步长为 `2`，自动修正为奇数。

说明：

- 彩色图会先转灰度。
- 输出为单通道梯度图。

### Laplacian 算子

使用 Laplacian 算子计算图片二阶导数。

参数：

- 核大小：步长为 `2`，自动修正为奇数。

说明：

- 彩色图会先转灰度。
- 输出为单通道图，对噪声敏感，建议先做平滑处理。

## 输出节点

### 预览

把输入图片集放到工作流右侧预览区。

操作：

- 多个预览节点会合并到同一个预览列表，可以用左右按钮切换查看；预览项会用画布上的 `预览 1`、`预览 2` 等节点名称标识来源。
- 双击预览图可以打开为单独的图像查看器。

### 保存图片

把输入图片集保存到指定目录。

参数：

- 输出目录：保存位置。
- 命名：文件名模板。
- 格式：输出格式。
- 覆盖已有文件：是否允许覆盖。

支持格式：

- `png`
- `jpg`
- `bmp`
- `tiff`
- `webp`

命名模板支持：

- `{stem}`：源文件名，不含扩展名。
- `{name}`：源文件名。
- `{index}`：图片序号。

如果模板没有扩展名，会自动追加当前选择的格式扩展名。

## 常见用法

### 对当前图片做灰度预览

1. 添加 `当前图片`。
2. 添加 `灰度`。
3. 添加 `预览`。
4. 连接：当前图片 -> 灰度 -> 预览。
5. 点击运行。

### 按颜色提取目标区域

1. 添加 `当前图片` 或 `图片目录`。
2. 添加 `HSV 范围阈值`。
3. 点击 `开始选点`，在图片上左键点击多个目标颜色点。
4. 按 `Ctrl+Z` 可以撤销最后一个点。
5. 右键结束选点，节点会自动生成 `HSV 下限` 和 `HSV 上限`。
6. 根据效果微调 `HSV 容差` 或手动修正 HSV 上限/下限。
7. 添加 `预览` 或 `保存图片`。
8. 连接并运行。

### 查看 HSV 三个通道

1. 添加 `当前图片`。
2. 添加 `色彩空间转换`，目标选择 `HSV`。
3. 添加 `通道分离`。
4. 分别把通道 1、通道 2、通道 3 接到预览或保存节点。

通道含义：

- 通道 1：H，色相。
- 通道 2：S，饱和度。
- 通道 3：V，亮度。

### 批量裁切并保存

1. 添加 `图片目录`。
2. 添加 `裁切`。
3. 在裁切节点里设置 X/Y/W/H，或使用自由框选。
4. 添加 `保存图片`。
5. 连接图片目录到裁切节点，再连接到保存图片节点。
6. 设置输出目录、命名模板和格式。
7. 点击运行。

## 注意事项

- 当前工作流不会记录每张图片的色彩空间状态，使用通道分离前需要知道上游节点输出的通道顺序。
- HSV 阈值节点可以自动从 BGR 转 HSV，除非输入本身已经是 HSV。
- 预览和保存节点是终端节点。工作流至少需要一个终端节点才能执行。
- 对大目录批量处理时，图片会被读取到内存中，图片数量或尺寸过大时会占用较多内存。
